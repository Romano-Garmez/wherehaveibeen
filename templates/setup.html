What you will need


  • Your own OwnTracks server
  • OwnTracks app on your phone
  • Your own OSRM server (maybe)
  • Your own domain (recommended, not required)

What you will do


Server Setup

First things first, get your own OwnTracks server set up. I'd offer to let you use mine, but I don't want to be responsible for your location data! The main things to look at are the official OwnTracks guide, and posts by real people trying to set it up

Official OwnTracks Guide Brian Douglass Guide

You need to get the server up and running, port forward as necessary so your server can be reached on the internet, then get your phone set up to start pinging your server!


OwnTracks App

Next up is to grab the OwnTracks app for iOS or Android. If all you want is location history, you can stop here. If you want to see what areas of the world (or west coast) you've explored so far, keep going!


OSRM Server (maybe optional)

If you're content with staying on the west coast, you can use my server for this. I host an Open Source Routing Machine server for Washington, Oregon, and California as those are the only areas I really visit. If you live elsewhere, you'll need to host your own. You can do that by following the readme linked below. Additionally, to use this you'll need to host this flask app yourself, as the OSRM server is hardcoded. I'll fix this in a future update.

OSRM Backend

Note! If you want more than one region, you can combine multiple .osm.pbf files into one to host with a tool called "osmium-tool".


Domain

You may have noticed this is all under the lovely "romangarms.com" domain. You don't need to do this, you could just enter your IP address into the url bar in your browser, but it makes things nice.
